# Fundacja Nowe Horyzonty — strona internetowa

To repozytorium zawiera pliki strony internetowej Fundacji Nowe Horyzonty, przygotowanej jako statyczna strona HTML do publikacji przez GitHub Pages.

## Zawartość repozytorium

Najważniejszy plik:
- `index.html` — główny plik strony

Dodatkowo w repozytorium mogą znajdować się:
- obrazy,
- logo,
- dokumenty do pobrania,
- dodatkowe pliki CSS lub JavaScript, jeśli strona będzie rozwijana.

## Jak uruchomić stronę

Po wgraniu pliku `index.html` do repozytorium i włączeniu GitHub Pages strona będzie dostępna online.

## Publikacja przez GitHub Pages

1. Wejdź do repozytorium na GitHub.
2. Otwórz `Settings`.
3. Przejdź do zakładki `Pages`.
4. Wybierz źródło publikacji:
   - branch: `main`
   - folder: `/root`
5. Zapisz ustawienia.
6. Po kilku minutach GitHub opublikuje stronę.

## Własna domena

Repozytorium może zostać podłączone do własnej domeny, np. `horyzonty.org`.

Po włączeniu GitHub Pages:
1. W `Settings -> Pages` wpisz własną domenę w polu `Custom domain`.
2. Ustaw odpowiednie rekordy DNS u operatora domeny.
3. Włącz `Enforce HTTPS`, gdy certyfikat będzie gotowy.

## Edycja strony

Aby zmienić treść strony:
1. Otwórz plik `index.html`.
2. Edytuj teksty, linki, zdjęcia lub sekcje.
3. Zapisz zmiany.
4. Wgraj zaktualizowany plik do repozytorium.

## Co warto uzupełnić

Przed publikacją finalnej wersji warto sprawdzić:
- dane kontaktowe,
- linki do dokumentów,
- sekcję aktualności,
- opisy projektów,
- politykę prywatności,
- stopkę i dane rejestrowe.
